Portfolio

Below you will find some of my projects that I have worked on. While working on projects, I focus on pixel perfection and make sure it's satisfactory from the user's prespective. The process of creating every project starts with client's usage discussion, then research and wireframing and then developing the idea.

Geo Fencing using Reactjs

This was my first Reactjs project so it holds a special place in my heart. It's built using Reactjs and Leaflet with OpenStreetMap data for mapping. It's a project to find if two users are in range of each other.

Jammu Basket

Jammu Basket is an grocery eCommerce website. I worked with a team of 4 and did the frontend part of the project. Website uses handlebars for the frontend and Nodejs (Express, MongoDB) as the backend. It's a fully functional eCommerce website.

Titiksha 2020

I was in charge of development for the website for the technical fest of SMVD University for the 2020 version. It's a static website which uses Google Sheets API for registration of events. I also did the SEO of the website and managed to rank 1 for multiple keywords. Apart from website, I also designed logo, Certificates, banners in Adobe XD and t-Shirt in Corel Draw for the event.

Punarjagran Samiti

This was a voluntary work for a Delhi, India based NGO which works for less privileged of the surrounding area. Website contains basic information about the NGO, a gallery to showcase their work, a contact page and donation details.

Tawi Mahal

This project was for one of the biggest banquet hall in the Jammu City. It is a static website with information about how to contact the hall, a gallery to showcase their services and a booking page where people can check the previous booking to plan their date. Registration page uses Google Sheets API to show details.
